## 🆘 Common Issues

**"Invalid credentials"**  
→ Double-check your email password

**"Emails going to spam"**  
→ Start with 5-10/day, warm up account  
→ Ask hosting provider about SPF/DKIM records

**"Backend not working"**  
→ Check running on port 8000

**"Connection timeout"**  
→ Verify por.problaq.co.za server is accessible  
→ Check port 465 not blocked by firewall
