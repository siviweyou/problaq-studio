## 🔄 User Journey

### 1. Discovery
```
User on homepage → Sees "Email Tool" in nav → Clicks
```

### 2. Setup
```
Scrolls to Section 06 → Reads intro → Gets App Password
```

### 3. Preparation
```
Opens spreadsheet → Copies emails → Pastes into form
```

### 4. Sending
```
Enters Gmail credentials → Clicks "Send to X" → Waits
```

### 5. Results
```
Sees success message → Checks stats → Notes failures
```

### 6. Follow-up
```
Saves results → Adds to tracking sheet → Schedules follow-ups
```
