# 🚀 Quick Start: Email Campaign System

## 1. Start Backend Server

```bash
cd backend
npm start
```

Server will run at: http://localhost:8000

---

## 2. Start Frontend Server

Open a new terminal window:

```bash
cd frontend
npm run dev
```

Frontend will run at: http://localhost:5173

---

## 3. Open Email Campaign Tool

Visit: **http://localhost:5173#email-campaign**

Or scroll down to **Section 06: Email Campaign Tool**

---

## 4. Get Gmail App Password (First Time Only)

1. Go to: https://myaccount.google.com/security
2. Enable **2-Step Verification**
3. Click **App Passwords**
4. Generate password for "Mail" → "Other (Problaq)"
5. **Copy the 16-character password**

---

## 5. Prepare Your Email List

Format (one per line):
```
email@domain.com, Name, Company Name
another@business.com, John Doe, Business Inc
contact@site.com
```

---

## 6. Send Campaign!

1. Paste your **Gmail address**
2. Paste your **Gmail App Password**
3. Paste your **email list**
4. Click **"Send to X Recipients"**
5. Watch the results! 🎉

---

## 📊 What You'll See:

- ✅ **Sent:** Successfully delivered
- ❌ **Failed:** Couldn't send (with error details)
- ⚠️ **Invalid:** Bad email format

---

## 💡 Pro Tips:

- Start with **10-20 emails** per day
- Send **Tue-Thu, 10am-3pm** for best results
- Include **name & company** for personalization
- Check **spam folder** on test emails first
- **Follow up** after 3-7 days
